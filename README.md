# ☕ Core Java Programs

This repository contains Core Java programs covering fundamental concepts of Java programming. It is designed for beginners, students, and interview preparation.

## 📚 Topics Covered

### 1. Java Basics
- Hello World
- Variables and Data Types
- Operators
- Input and Output
- Type Casting

### 2. Control Statements
- If, If-Else
- Nested If
- Switch Case
- Loops (for, while, do-while)
- Break and Continue

### 3. Arrays
- 1D Arrays
- 2D Arrays
- Array Operations
- Searching Algorithms
- Sorting Algorithms

### 4. Strings
- String Methods
- StringBuffer
- StringBuilder
- Palindrome Check
- Anagram Programs

### 5. Methods
- Method Creation
- Method Overloading
- Recursion

### 6. Object-Oriented Programming (OOP)
- Classes and Objects
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation



## 🛠️ Technologies Used

- Java
- JDK 8+
- IntelliJ IDEA / Eclipse / VS Code

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/core-java-programs.git
```

### Navigate to the Project

```bash
cd core-java-programs
```

### Compile a Program

```bash
javac ProgramName.java
```

### Run a Program

```bash
java ProgramName
```
